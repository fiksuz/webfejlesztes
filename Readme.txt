##WebDev Website
A simple website created using VS Code.

##Necessary Links:
-Git repository: https://github.com/fiksuz/webfejlesztes

##Installation:
-Download or clone the repository / download the zip file containing the program to your computer.
-Open the project folder in VS Code.

##Running:
-Install the Live Server extension in Visual Studio Code (if you haven't already done it yet).
-Right-click on the index.html file.
-Choose the "Open with Live Server" option from the menu.

After this, the page will automatically open in your default browser.

##Usage:
Since the purpose of the website is mainly for demonstration, it's straightforward to use. Simply navigate between pages using the menu options and other buttons.
